Tutorial

The Left box above (canvas) is the FSM designer.  Here's how to use it:

Adding States

Action:

double-click on the canvas.

Description:

Create a single state and add cursor into it.

Adding Links

Action:

Shift + Drag on the canvas.

Description:

Create an edge start from the cursor location . If cursor is on the state, then set link from that state.

Adding States

Move:

Drag states or links over the canvas.

Description:

move states/links around.

Accept State

Action:

double-click on the created state to make it Accept State or remove if it already is.

Description:

add and remove accept state.

Delete

Action:

Click on the state/link and press delete.

Description:

Can be used for deleting object on canvas.

Modify Props

Action:

click on the small square near the specific field to edit it.

Description:

can be used to add optional properties to the FSM.
